# Best Shot App

Welcome to Best Shot! This is a Football game where users can guess real-world match outcomes and play with friends. This is NOT a betting application.

This repository contains the codebase for the Web, Android, and iOS applications, built with React Native and Expo.

## Project Goals

- Develop a single, integrated codebase for all three platforms (Web, Android, iOS).
- Consume an existing API for game data and logic.
- Provide a fun and engaging user experience.
