# React Native Mobile App Development Course

This folder contains documentation for each lesson in our React Native mobile app development course.

## Course Overview

- **Duration**: 12 weeks
- **Schedule**: Weekly 2-hour sessions on Fridays
- **Focus**: Building a cross-platform (Web, iOS, Android) version of Best Shot
- **Architecture**: Hybrid DDD + Feature-First approach

## Lesson Structure

Each lesson follows this format:
1. **Review** (15 min) - Quick review of previous concepts and homework
2. **New Concepts** (30 min) - Introducing new mobile development topics
3. **Coding Session** (60 min) - Hands-on implementation
4. **Homework & Preview** (15 min) - Tasks for the next week
